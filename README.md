    - for loop : when you have a specific number to repeat the same process, you can use a for loop.   
    ```python

        print(each)
    ```
    - while loop : when you know the condition must be satisfied, you can use a while loop. 
    ```python
    while(True):
        print('print this endlessly')
    ```
